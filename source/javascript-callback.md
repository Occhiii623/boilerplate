### JavaScript(コールバック関数)
今日は手始めに、ProgateでJavaScriptのコールバック関数について復習。

ちょっと概念がわかりにくいところ。

```javascript
// (2) callback()によって、次に引数に渡した関数が呼び出されて実行される
const printName = () => {
  console.log("私の名前は、Mayumiです。");
 };
 
 // (1) 関数callがまず先に実行される
 const call = (callback) => {
  console.log("コールバック関数を呼び出します。");
  callback();
 };
 
 call(printName);
```
